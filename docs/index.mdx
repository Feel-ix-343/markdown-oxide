# Markdown-Oxide

**A powerful PKM Language Server that brings Obsidian-like features to your favorite text editor.**

Markdown-Oxide is a [PKMS](/reference/pkms) that composes with your favorite text-editor through the [LSP](/reference/lsp). While other [PKMS](/reference/pkms) implementations include their own text-editors, markdown-oxide is *unbundled*: it leaves text-editing to a dedicated text-editor and focuses solely on robust, performant knowledge management.

## Why Markdown-Oxide?

<AccordionGroup>
  <Accordion title="Use Your Favorite Editor" icon="keyboard">
    You've spent days configuring your text editor. Why give it up? Markdown-Oxide works with any LSP-compatible editor, letting you keep your workflow while gaining powerful PKM features.
  </Accordion>
  
  <Accordion title="Obsidian-Compatible" icon="link">
    Highly compatible with Obsidian vaults. Import your settings from `.obsidian/` configuration files and use familiar syntax like wikilinks, tags, and daily notes.
  </Accordion>
  
  <Accordion title="Fast & Performant" icon="bolt">
    Built in Rust for speed and reliability. Handles large vaults with thousands of notes without breaking a sweat.
  </Accordion>
  
  <Accordion title="Editor Support" icon="code">
    Best supported on Neovim, with excellent support for VSCode, Helix, and Zed. Works with any editor implementing the Language Server Protocol.
  </Accordion>
</AccordionGroup>

## Quick Start

Choose your editor to get started:

<Tabs>
  <Tab title="Neovim">
    Install via your preferred method:
    ```bash
    # Cargo
    cargo install --locked --git https://github.com/Feel-ix-343/markdown-oxide.git markdown-oxide
    
    # Arch Linux
    pacman -S markdown-oxide
    
    # Homebrew
    brew install markdown-oxide
    ```
    
    Then configure nvim-lspconfig with dynamic registration enabled. [Full setup guide →](/setup-instructions#neovim)
  </Tab>
  
  <Tab title="VSCode">
    Install the [Markdown Oxide extension](https://marketplace.visualstudio.com/items?itemName=FelixZeller.markdown-oxide) from the VSCode marketplace.
    
    The extension will automatically download and manage the language server binary. [Full setup guide →](/setup-instructions#vscode)
  </Tab>
  
  <Tab title="Zed">
    Install the Markdown Oxide extension from Zed's extension marketplace.
    
    The extension will automatically download and manage the language server binary. [Full setup guide →](/setup-instructions#zed)
  </Tab>
  
  <Tab title="Helix">
    Install the binary to your PATH:
    ```bash
    # Cargo
    cargo install --locked --git https://github.com/Feel-ix-343/markdown-oxide.git markdown-oxide
    
    # Arch Linux
    pacman -S markdown-oxide
    ```
    
    Helix will automatically detect and use the language server. [Full setup guide →](/setup-instructions#helix)
  </Tab>
</Tabs>

## Is Markdown-Oxide Right for You?

Markdown-oxide is perfect if you:

- Have a favorite text editor that supports LSP and want to keep using it
- Want Obsidian-like PKM features without being locked into a specific app
- Need a fast, performant solution for large markdown vaults
- Value the Unix philosophy of composable, focused tools

If these resonate with you, [get started now](/setup-instructions)!
